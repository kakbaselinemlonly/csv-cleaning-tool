# CSV Data Cleaning Tool

A simple Streamlit web application for basic CSV data cleaning.

## Features

- Upload a CSV file
- Preview the original dataset
- Check dataset shape, duplicate rows, and missing values
- Remove duplicate rows
- Handle missing values by dropping rows, filling with 0, or filling with N/A
- Standardize column names
- Download the cleaned CSV file

## Tech Stack

- Python
- pandas
- Streamlit
